Bring your own everything

  • • API key lives in your browser; sent only in a request header
  • • Never written to a database or a log
  • • Resume rendering & ATS check run fully client-side
  • • Works offline / guest; Supabase sync is optional

Two engines

  • • Key-free deterministic path: autofill, scoring, ATS check
  • • BYOK LLM path: 5 providers (Anthropic, OpenAI, Gemini, MiMo, custom)
  • • Truth-only tailoring — one lane, no invented skills or metrics
  • • serper.dev web search for company research

The learning loop

  • • Tailoring surfaces honesty gaps + defensible strengths
  • • Interview retros log wins, risks and skill gaps per phase
  • • Both feed the Knowledge Bank (suggest-and-confirm, deduped)
  • • Which sharpens the next resume and interview prep
